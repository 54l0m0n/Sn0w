# Sn0w
Tool development to monitoring and capture packets in local networks.

# Install

 $ git clone https://github.com/54l0m0n/Sn0w.git

 $ chmod +x snow.py

 $ ./snow.py

# Functionality
Capture packets:
- SSH
- TELNET
- FTP
- HTTP
- SMTP 
- POP3 
- IMAP
- SNMP 
- IRC

# Tested
- Kali Linux 2.0
- Debian 8.0

#Requirements
- Python 2.7
